Правила сервера

  • Уважение к другим игрокам: Любые оскорбления, троллинг и дискриминация запрещены
  • Читерство: Использование читов, модов дающих преимущество, дюпов - бан
  • Гриферство: Разрешен только в специальных PvP зонах
  • Реклама: Запрещена любая реклама других серверов
  • Баги: Обнаруженные баги нужно сообщать администрации, использование запрещено
  • Спам: Не спамьте в чат, используйте для этого специальные каналы
  • 18+ контент: Запрещены любые отсылки к взрослому контенту

Как подключиться

🎮 Подключение к серверу

Требования:

  • Minecraft Java Edition версии 1.21.11 или ниже
  • Стабильное интернет-соединение
  • Официальный клиент Minecraft (без читов) (или если нет лицензии можно и пиратский)

IP-адрес сервера:

IP: hwpro.online

Нажмите кнопку выше чтобы скопировать IP-адрес

Инструкция по подключению:

  1. Запустите Minecraft Java Edition 1.21.11 или ниже
  2. В главном меню нажмите "Сетевая игра"
  3. Выберите "Добавить сервер"
  4. Вставьте IP-адрес нашего сервера
  5. Нажмите "Готово" и подключитесь

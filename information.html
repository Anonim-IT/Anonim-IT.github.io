Информация о проекте

Здесь будет размещена подробная информация о проекте Anonim Community.

Наш проект включает несколько ключевых модулей, которые развиваются с учетом потребностей сообщества:

  • AnonimCommunity Server — Наш официальный Discord сервер.
  • AnonimForum — Форум проекта, где обсуждаются новости и вопросы, связанные с проектом.
  • AnonimWiki — Вики-страница проекта с подробной информацией о каждом модуле.
  • AnonimWorld — Сайт с будущими проектами и планами, которые будут реализованы в рамках Anonim Community.
